1. New Linux System w/ Aptitude PKG Manager

1.1. Change Console Resolution on Headless Ubuntu System (Q&A)

1.1.1. Open /etc/default/grub and change/add the following

GRUBGFXMODE=1920x1080x32 GRUBGFXPAYLOAD=1920x1080x32

1.1.2. Open /etc/grub.d/00header and Change the following

if [ "x\({GRUB_GFXMODE}" = "x" ] ; then GRUB_GFXMODE=1920x1080x32 ; fi if [ "x\){GRUBGFXPAYLOAD}" = "x" ] ; then GRUBGFXPAYLOAD=1920x1080x32 ; fi

1.1.3. Add the following

set gfxmode=\({GRUB_GFXMODE} set gfxpayload=\){GRUBGFXPAYLOAD}

1.1.4. sudo update-grub2

1.2. Installing Vundle for vim

1.2.1. Vundle

git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim

1.2.2. lsd

git clone https://github.com/Peltoche/lsd cargo install lsd export PATH=/home/USERNAME/.cargo/bin:$PATH

1.2.3. Only if using clean vimrc

set nocompatible " be iMproved, required filetype off " required

" set the runtime path to include Vundle and initialize set rtp+=~/.vim/bundle/Vundle.vim call vundle#begin()

1.3. apt packages

sudo apt install npm python3.10 base-devel linux-headers-$(uname -r) emacs vim yarn thefuck tmux btop bat
